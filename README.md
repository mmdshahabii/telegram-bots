# ربات‌های تلگرام با پایتون 🤖

این مخزن شامل پروژه‌های تمرینی منه برای یادگیری ساخت ربات تلگرام با پایتون.

## پروژه‌ها

| فایل | توضیح |
|------|--------|
| `my_first_bot.py` | اولین ربات ساده (echo) |
| `button_bot.py` | ربات دکمه‌ای (InlineKeyboard) |
| `coffee_bot.py` | ربات سفارش قهوه |
| `support_bot.py` | ربات پشتیبانی |
| `register_bot.py` | ربات ثبت‌نام چند مرحله‌ای |
| `database_bot.py` | ربات با ذخیره‌سازی در JSON |
| `task_manager_bot.py` | ربات مدیریت تسک |
